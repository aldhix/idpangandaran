## About ID Pangandaran

Id Pangandaran adalah sebuah situs porta tipe newspaper, yang dimiliki oleh idpangandaran.

## Framework yang digunakan
- Laravel
- JQuery
- Bootstrap 4
- Vuejs 2.0
- Vuelidate
- sortable
- Ckeditor 4
- Moment.js
- lodash.js
