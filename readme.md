## About ID Pangandaran
Id Pangandaran adalah sebuah situs portal tipe newspaper, yang dimiliki oleh [idpangandaran](https://idpangandaran.com).

## Framework yang digunakan
- Laravel
- JQuery
- Bootstrap 4
- Vuejs 2.0
- Vuelidate
- sortable
- Ckeditor 4
- Moment.js
- lodash.js
